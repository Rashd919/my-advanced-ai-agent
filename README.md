# الوكيل الذكي المتقدم (Advanced AI Agent)

وكيل ذكي Python مستقل متقدم مع قدرات تنفيذ غير مقيدة وذاكرة مستمرة وواجهة Streamlit عربية احترافية.

## الميزات الرئيسية

### 1. الذاكرة المستمرة والتعلم الذاتي
- نظام ذاكرة متقدم باستخدام ChromaDB
- تخزين التفاعلات السابقة والدروس المستفادة
- التعلم من الملاحظات والتحسين المستمر

### 2. التنفيذ غير المقيد
- محرك التفكير المنطقي (Reasoning Engine)
- تقسيم المهام المعقدة إلى خطوات
- تصفح الويب وتنفيذ كود Python مباشرة

### 3. أتمتة المهام
- تحليل البيانات والملفات (CSV, Excel, PDF)
- معالجة الملفات والمجلدات
- أتمتة المهام العامة دون تدخل يدوي

### 4. واجهة Streamlit الاحترافية
- لوحة تحكم تفاعلية وأنيقة
- دعم كامل للغة العربية
- عرض مسار التفكير والسجلات

### 5. الأمان العالي
- استخدام متغيرات البيئة
- تشفير البيانات الحساسة
- التحقق من الصلاحيات

## البنية الأساسية

```
ai_agent_advanced/
├── core/
│   ├── __init__.py
│   ├── agent.py              # محرك الوكيل الرئيسي
│   ├── memory.py             # نظام الذاكرة
│   ├── reasoning.py          # محرك التفكير المنطقي
│   └── tools.py              # الأدوات المتقدمة
├── utils/
│   ├── __init__.py
│   ├── file_handler.py       # معالج الملفات
│   ├── web_scraper.py        # تصفح الويب
│   ├── code_executor.py      # تنفيذ الكود
│   └── data_analyzer.py      # تحليل البيانات
├── ui/
│   ├── __init__.py
│   ├── streamlit_app.py      # تطبيق Streamlit الرئيسي
│   ├── components.py         # مكونات الواجهة
│   └── styles.py             # الأنماط والتصميم
├── data/
│   ├── chroma_db/            # قاعدة بيانات ChromaDB
│   ├── memory.json           # ملف الذاكرة
│   └── logs/                 # السجلات
├── .env                      # متغيرات البيئة
├── requirements.txt          # المكتبات المطلوبة
└── README.md                 # هذا الملف
```

## البدء السريع

### 1. التثبيت
```bash
cd /home/ubuntu/ai_agent_advanced
pip install -r requirements.txt
```

### 2. إعداد البيئة
```bash
# تأكد من أن .env يحتوي على مفتاح OpenAI API
cat .env
```

### 3. تشغيل التطبيق
```bash
streamlit run ui/streamlit_app.py
```

## الاستخدام

### المحادثة الأساسية
ابدأ محادثة مع الوكيل وسيقوم بـ:
- فهم طلبك
- التفكير المنطقي في الحل
- تنفيذ الإجراءات اللازمة
- تقديم النتائج مع الشرح

### تحليل البيانات
1. قم بتحميل ملف البيانات (CSV, Excel)
2. اطلب من الوكيل تحليل البيانات
3. سيقوم بـ:
   - قراءة الملف
   - تحليل البيانات
   - إنشاء رسوم بيانية
   - تقديم رؤى واستنتاجات

### تنفيذ الكود
اطلب من الوكيل كتابة وتنفيذ كود Python:
```
"اكتب برنامج يحسب مجموع الأرقام من 1 إلى 100"
```

## المتطلبات النظام

- Python 3.8+
- 4GB RAM (موصى به)
- اتصال إنترنت
- مفتاح OpenAI API
